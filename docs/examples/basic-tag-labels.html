CSSコード

/* 基本スタイル */
.tag {
    display: inline-block;
    padding: 4px 8px;
    font-size: 12px;
    font-weight: 500;
    line-height: 1;
    border-radius: 12px;
    text-decoration: none;
    margin: 2px;
    white-space: nowrap;
    transition: all 0.2s ease;
    cursor: pointer;
}

.tag:hover {
    transform: translateY(-1px);
    box-shadow: 0 2px 5px rgba(0,0,0,0.2);
}

/* カラーバリエーション */
.tag-primary { background-color: #007acc; color: white; }
.tag-secondary { background-color: #6c757d; color: white; }
.tag-success { background-color: #28a745; color: white; }
.tag-danger { background-color: #dc3545; color: white; }
.tag-warning { background-color: #ffc107; color: #212529; }
.tag-info { background-color: #17a2b8; color: white; }
.tag-light { background-color: #f8f9fa; color: #212529; border: 1px solid #dee2e6; }
.tag-dark { background-color: #343a40; color: white; }

HTMLの使用方法

<!-- 基本的な使用法 -->
<span class="tag tag-primary">Primary</span>
<span class="tag tag-success">Success</span>
<span class="tag tag-danger">Danger</span>

<!-- サイズ指定 -->
<span class="tag tag-primary tag-sm">Small</span>
<span class="tag tag-primary tag-lg">Large</span>

<!-- アウトライン -->
<span class="tag tag-outline tag-primary">Outline</span>

<!-- グラデーション -->
<span class="tag tag-gradient-blue">Gradient</span>

<!-- 角丸なし -->
<span class="tag tag-primary tag-square">Square</span>

<!-- 組み合わせ例 -->
<span class="tag tag-outline tag-success tag-lg">大きなアウトライン</span>

<!-- 実際の使用例 -->
<p>プロジェクト: ECサイト <span class="tag tag-success">進行中</span></p>
<p>技術: <span class="tag tag-gradient-blue">React</span> <span class="tag tag-info">TypeScript</span></p>

アクセシビリティとベストプラクティス

  • 色の組み合わせ: 十分なコントラスト比を確保（WCAG 2.1 AA準拠）
  • ホバー効果: マウスオーバー時の視覚的フィードバック
  • レスポンシブ: 小さな画面でも読みやすいサイズ設定
  • セマンティクス: 意味のあるクラス名の使用
  • パフォーマンス: 軽量なCSS設計

推奨される使用方法

  • ステータス表示にはsuccess, warning, dangerを使用
  • カテゴリ分けにはprimary, secondary, infoを使用
  • 特別な要素にはnew, hot, trendingを使用
  • 技術スタックにはグラデーションスタイルを活用
