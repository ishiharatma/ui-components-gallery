CSSコード例

方法1: ::after疑似要素

/* 基本スタイル */
.tag {
    display: inline-block;
    padding: 6px 12px;
    font-size: 13px;
    font-weight: 600;
    border-radius: 16px;
    background: linear-gradient(135deg, #FF9800 0%, #FFC107 100%);
    color: white;
    margin: 4px;
}

/* 各カテゴリに絵文字を自動追加 */
.tag-aws-ai::after {
    content: "🤖";
    margin-left: 6px;
}

.tag-aws-cloud-ops::after {
    content: "☁️";
    margin-left: 6px;
}

.tag-aws-containers::after {
    content: "📦";
    margin-left: 6px;
}

.tag-aws-data::after {
    content: "🛢️";
    margin-left: 6px;
}

/* 他のカテゴリも同様に... */

方法2: ::before疑似要素

/* 絵文字を前に配置 */
.tag-aws-ai-before::before {
    content: "🤖";
    margin-right: 6px;
}

方法3: CSS変数使用

/* CSS変数を使った柔軟な方法 */
.tag-custom::after {
    content: var(--emoji, "");
    margin-left: 6px;
}

/* HTMLで使用 */
<span class="tag tag-custom" style="--emoji: '🤖';">AIエンジニアリング</span>

HTMLの使用方法

<!-- 方法1: クラス指定だけで自動的に絵文字が追加される -->
<span class="tag tag-aws-community-builders tag-aws-ai">AIエンジニアリング</span>
<span class="tag tag-aws-community-builders tag-aws-serverless">サーバーレス</span>
<span class="tag tag-aws-community-builders tag-aws-security">セキュリティ</span>

<!-- 方法3: CSS変数で動的に指定 -->
<span class="tag tag-aws-community-builders tag-custom" style="--emoji: '🤖';">AIエンジニアリング</span>

<!-- 実際の使用例 -->
<p>田中さん <span class="tag tag-aws-community-builders tag-aws-ai">AIエンジニアリング</span></p>

メリット

  • 一貫性: すべてのラベルで統一されたスタイル
  • 保守性: 絵文字の変更は1箇所のCSS修正で完了
  • 簡潔性: HTMLには絵文字を書く必要がない
  • 自動化: クラス名だけで絵文字が自動追加される
  • レスポンシブ: 絵文字も含めてホバーエフェクトが適用される
