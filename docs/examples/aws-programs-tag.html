色分けの理由

  • 🟢 Jr. Champion（緑系）: 成長・新緑・フレッシュさを表現
  • 🔵 Top Engineers（青系）: 技術的な深さ・信頼性・プロフェッショナリズム
  • 🟠 Community Builders（オレンジ系）: 温かさ・コミュニティ・活発さ
  • 🟣 AWS Hero（紫・ピンク系）: 最高位の威厳・特別感・エクスクルーシブ

    display: inline-block;
    padding: 6px 12px;
    font-size: 13px;
    font-weight: 600;
    line-height: 1;
    border-radius: 16px;
    text-decoration: none;
    margin: 4px;
    white-space: nowrap;
    transition: all 0.3s ease;
}
